### Team

### Beschreibung
Geplant ist ein klassissches Arkade Spiel mit mehreren autospuren.
Das spiel wird als einzelspieler laufen, mit einem Highscore-Ergebnis. Umgesetzts soll es mit Java in der online-ide laufen.
